# Важность Heroku для разработчика

Ключевая особенность облачного сервиса [Heroku](https://www.heroku.com/) состоит в том, что запуск сервиса в работу осуществляется публикацией репозитария проекта на определённый URL, используя протокол git. Это обеспечивает очень низкий порог вхождения для запуска web-приложения в публичный доступ.
