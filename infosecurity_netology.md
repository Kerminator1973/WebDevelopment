# Информационная безопасность

Intrusion Prevention System (IPS) - система предотвращения вторжений, защитное решение, предназначенное для обнаружения и пресечения потенциально опасной активности в компьютерной сети или на отдельном устройстве.

Межсетевой экран (Firewall) - обеспечивает защиту периметра, фильтрует трафик по заданным правилам. Анализирует заголовки сетевых пакетов и сравнивает их параметры с набором правил (ruleset). Проверяемые параметры:

- IP‑адрес источника и назначения
- порт источника и назначения
- используемый протокол (TCP, UDP, ICMP и др.)
- флаги протокола
- состояние соединения

Осуществляет **контроль сессий** (stateful inspection):

- фиксируют установление сеанса связи в специальной таблице
- пропускают только пакеты, относящиеся к уже установленным соединениям
- отбрасывают пакеты, имитирующие пакеты завершённых сессий

Посредством **глубокого анализа пакетов** (Deep Packet Inspection, DPI) выявляет:

- вредоносный код
- подозрительные шаблоны данных
- попытки эксплуатации уязвимостей

Политики безопасности определяют:

- какой трафик разрешён
- какой трафик запрещён
- какие действия выполнять с подозрительными пакетами

Ключевым для ИБ является система аутентификации.

Центр мониторинга безопасности (SOC = Security Operation Center) - управляющая консоль для контроля ИБ.

## Организация удалённой работы

Remote Access VPN - подключение к корпоративной сети. У удалённого сотрудника должны быть Антивирус и VPN-доступ. Одна из ключевых атак злоумышленников - DNS redirect. Рекомендуется устанавливать: Anti-Malware, DNS Security, Personal Firewall, Posture Assessment.

>Kaspersky Secure Connection - это не VPN для доступа к сервисам внешнего интернета, а защита "DNS Security" + VPN.
