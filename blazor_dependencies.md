# Работа с зависимостями

При работе с большими проектами может потребоваться понять, каким образом в проект попала Transitive-зависимость. Transitive Packages (транзитивные пакеты) в Visual Studio — это пакеты NuGet, которые подключаются в проект не напрямую, а через зависимости от других пакетов (прямых зависимостей). 

Начиная с .NET Core 8, можно использовать команду `why`:

```csharp
dotnet nuget why ServicePartners.Server.csproj AutoMapper
```
