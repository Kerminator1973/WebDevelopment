# Цель решения

Веб-интерфейс для просмотра и усправления списком с пагинацией, поиском и сортировкой.

Оба файла реализуют отображение таблицы с данными, загружаемыми с API посредством http-запроса. Все данные для таблицы должны быть загружены в память браузера. 

Ключевые пользовательские сценарии:

- просмотр данных
- поиск по частично введённому тексту из ячеек таблицы
- сортировка по щелчку на заголовке таблицы

## Требования к поведению веб-интерфейса

Пользовательский интерфейс должен так же содержать кнопки "Добавить", "Изменить", "Удалить". 

Кнопки "Изменить" и "Удалить" должны быть доступны только в том случае, если в таблице есть один выбранный элемент.

При нажатии на кнопку "Добавить" должен ототбражаться модальный диалог с полями ввода, соответствующими колонкам таблицы. Значения в полях модального даилога должны быть пустыми.

При нажатии на кнопку "Изменить" должен ототбражаться модальный диалог с полями ввода, соответствующими колонкам таблицы. Значения в каждом поле должны соответствовать значениям взятым из текущей выбранной строке таблицы.

При нажатии на кнопку "Удалить" должен ототбражаться модальный диалог с кнопками "Подтвердить" и "Отменить".

При подтверждении действия в модальных диалогах должен быть выполнен запрос к API, выполняющий соответсвующее кнопаке действие.

## Технические требования

- Библиотека для отображения библиотеки DataTables.NET 2.3.x
- Базовая библиотека jQuery 3.7.x
- Библиотека стилистического оформления: Bootstrap 5.1.x
- Все библиотеки должны быть загружены в проект. Использование CDN не допускается
- Формат получения данных от API: JSON через AJAX-запрос
- Сортировка и поиск должны осуществляться без необходимости взаимодействия с API
- Браузерная совместимость: Google Chrome, Yandex.Browser

## Структура HTML‑документа

- Базовая разметка (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Подключение CSS и JS (DataTables, jQuery, Bootstrap)
- Секция `<script>` для инициализации DataTables (пустая заглушка с комментарием "сюда вставляется JS‑код")

## Спецификация таблицы DataTables

Таблица должна содержать поле с идентификаторов записи в базе данных. Поиск по этому полю должен быть запрещён:

```js
datatable_settings["columns"] = [
    {
        "data": "id",
        className: "dt-body-center",
        "searchable": false // Не ищем по номеру идентификатора
    },
];
```

Сортировка по умолчанию должна осуществляться по первому полю.

По умолчанию, на странице таблицы должно отображаться не 14 строк.

## JavaScript‑код (клиентская логика)

- Инициализация DataTables с полным набором опций
- Обработчики AJAX‑запросов (успех/ошибка)
- Вспомогательные функции (форматирование ячеек, кнопки действий)
