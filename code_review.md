# Автоматизация выполнения Code Review

Есть разные категории задач, для автоматизации **code review**:

- используя IDE-bases инструментальные средства. Инструменты предоставляют real-time feedback: синтаксическая подсвета, предложения по улучшению кода
- используя средства на базе технологии Git. Инструменты встроены в GitHub, GitLab, BitBucket, и т.д. Срабатывают при выполнении commit-а в репозитарий
- специализированные сайты и web-приложения. Используют разные инструменты: linter-ы (например, ESLint, Pylint), системы статического анализа кода (например, SonarQube) и AI-powered системы анализа кода (например: DeepCode и Codacy)
