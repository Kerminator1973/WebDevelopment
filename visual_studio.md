# Кастомизация Visual Studio 2022 и 2026

В 2026 году в России рекомендуется сразу же отключить:

- ASP.NET и разработка web-приложений - Copilot (не работает в РФ)
- Live Share
- Разработка классических приложений .NET - Blend for Visual Studio
- ML.NET Model Builder - есть более интересные альтернативы

Включить:

- ASP.NET и разработка web-приложений - поддержка времени разработки в IIS
